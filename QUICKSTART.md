# 🚀 QUICKSTART

## In 3 Befehlen zur laufenden App:

```bash
# 1. In Projekt-Ordner
cd ~/devel/trading-app

# 2. Deployen
./deploy.sh

# 3. Im Browser öffnen
# Das deploy.sh Script fragt ob Port-Forward starten
# Dann: http://localhost:8080
```

## Alternativ mit Makefile:

```bash
make deploy
make port-forward
# Öffne: http://localhost:8080
```

## Das wars! 🎉

Die App läuft jetzt in deinem Kind Kubernetes Cluster
und zeigt live Bitcoin Trading-Signale.
